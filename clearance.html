How to read it

A deterministic decision. Agents that only draft. A human who ratifies.

The decision is deterministic
The binding cleared/routed result is a pure function over the product and a pinned, cited catalog version. No model touches it, and the same inputs reproduce the same determination for an examiner.
Agents only draft
Where an obligation has no control, the knowledge-plane agents draft a cited one — but it does nothing until a person ratifies it into the catalog, gated on a recall check so coverage can't regress.
A challenger can refute
An adversarial reviewer stress-tests every clear across distinct lenses. If it refutes one — like the EU Art. 13 row above — it escalates to a human. It adds caution; it never makes the call.
